# AI 뉴스 교육 연결성 규칙 (축G relevance scoring)

> 최신 AI 뉴스를 "우리 아이 교육에 무슨 뜻인가"로 번역하는 필터. 최신성=입구, 교육 해석=본문.

## 연결성 점수 (0~5)
| 점수 | 기준 | 처리 |
|---|---|---|
| 0 | 기업 실적/투자/주가 중심. 교육 연결 없음 | **hold_no_education_link** |
| 1 | 일반 AI 신제품. 교육 연결 약함 | 대개 hold |
| 2 | 학생의 정보 검색·콘텐츠 생성에 간접 영향 | 관찰(등록 선택) |
| 3 | 학교/숙제/가정 학습에 연결 가능 | **update_queue 등록 가능** |
| 4 | 아이 안전·개인정보·감정·의존에 영향 가능 | 등록 + **caution 상향** |
| 5 | 미성년자·교육·AI 튜터·AI companion·학교 사용과 직접 관련 | **우선 검토** |

## 판정 흐름
```
AI 최신 뉴스
 → 아이 교육과 연결되는가?  (아니오 → hold_no_education_link)
 → 부모가 알아야 할 변화인가?
 → 가정 규칙/학교 교육/감정 보호/진로 이해에 영향?
 → 점수 3+ → update_queue(콘텐츠 후보)  / 3미만 → 보류
```

## 번역 예시 (뉴스 → 우리 관점 → 부모 질문)
- **중국 감정 로봇**: 뉴스=감정형 AI 로봇 등장 → 관점=AI가 학습 도구를 넘어 감정·친밀감·관계로 → 질문=AI 친구/선생님/상담자를 아이에게 어디까지? (점수 4, ai_companion_issue) → UQ-06
- **NVIDIA 등 하드웨어**: 뉴스=AI 칩·로봇·온디바이스·에이전트 발전 → 관점=아이 환경(폰·장난감·교실·로봇)에 AI 기본 탑재 가능 → 질문=앱 하나가 아니라 생활 환경 전체로 이해시켜야? (점수 3, ai_device_environment)
- **Google/Meta 모델·AI 안경·검색·생성**: 뉴스=새 모델·에이전트·이미지/영상 생성 → 관점=아이가 정보 찾고 숙제하고 만들고 대화하는 방식 변화 → 질문=AI 답 확인법·AI 생성물 구분법을 어떻게 가르칠까? (점수 3, technology_shift)

## 사용 원칙 (못 박음)
- 빅테크 뉴스는 콘텐츠 **메인 아님**. 투자·홍보·신제품 소개 금지.
- 기업 발표 = 기술 변화 근거로만. **교육 정책 fact 수정 근거 아님**(신뢰 official보다 낮음, 과장 가능).
- 미성년자·감정·친밀감·의존 뉴스 = caution 상향, requires_anonymization.
