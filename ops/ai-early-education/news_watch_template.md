# News Watch — 뉴스/언론 기록 (축B)

> 뉴스 = **사회 반응·논란·변화 감지.** 사실 근거 아님 → 공식(축A) 확인 전 사이트 사실 수정 금지.
> 파일명 예: `news_watch/2026-07-05_<slug>.md`

## 기사 1
- outlet(매체):
- headline / url / publish_date:
- country_or_region:
- what_changed(무슨 일):
- signal_type: controversy_signal / policy_signal
- parent_relevance(부모에게 주는 의미):
- needs_official_verification: true (사실은 축A에서 확인)
- minor_data_in_article: 있음/없음 → 있으면 익명화/일반화
- update_queue 등록: controversy_signal / policy_signal / source_lead

## 원칙
- 단일 보도를 전체 흐름처럼 과장하지 않는다.
- 미성년자 이름/얼굴/학교 노출 기사는 직접 인용/캡처 금지 → 일반화.
- 사실 반영은 공식 발표·2개 이상 교차확인 후.
