# Industry Watch — AI 산업/빅테크 기록 (축G)

> 빅테크/AI 산업 변화는 **메인 뉴스가 아니다.** "이게 아이 교육과 무슨 관련?"만 본다.
> 투자·주가·기업홍보·신제품 소개 채널 금지. 교육 연결성 없으면 hold_no_education_link.
> 파일명 예: `industry_watch/2026-07-05_<slug>.md`

## 항목 1
- company / lab:
- headline / url / date:
- what_happened(기술 변화 요지):
- source_category: ai_company_news / ai_lab_release / ai_model_release / ai_robotics / ai_hardware / ai_agent / ai_search / ai_companion / ai_safety_company / ai_education_product
- trust_level: B_company_official / C_industry_media / D_creator_analysis / E_unverified_hype
- **education_relevance_score(0~5)**: (규칙=ai_news_relevance_rules.md)
- 연결 영역(체크): 학습 도구 / 검색·정보 확인 / 숙제·평가 / 감정·친밀감·의존 / AI 로봇·장난감·기기 / 영상·이미지 생성 / 개인정보·안전 / 미래 직업·진로
- our_translation(우리 관점 한 줄):
- parent_question(부모가 던질 질문):
- hype_risk: low/medium/high (과장 가능성)
- minor/privacy caution:
- update_queue 등록: industry_signal / technology_shift / parent_awareness_topic / future_risk_signal / child_safety_signal / ai_companion_issue / ai_device_environment / hold_no_education_link

## 원칙
- 기업 발표는 official_policy보다 낮은 신뢰(B_company_official). **정책 fact_source로 쓰지 않음**(기술 변화 근거로만).
- education_relevance_score < 3 → hold_no_education_link.
- 미성년자·감정·친밀감·의존 관련은 caution 상향(requires_anonymization).
- 고연결 태그(ai_robot·ai_companion·ai_agent·ai_search·ai_glasses·ai_video_generation·ai_tutor·ai_device·on_device_ai)는 별도 추적.
