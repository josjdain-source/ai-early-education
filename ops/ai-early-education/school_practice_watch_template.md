# School Practice Watch — 학교 현장/교사 수업 기록 (축D)

> 학교 현장 = **실제 수업 방식 참고.** 단일 사례를 일반화하지 않는다.
> 파일명 예: `school_practice_watch/2026-07-05_<slug>.md`

## 사례 1
- source(학교/교사/기관, 특정 금지 → '한 학교'):
- url / date / country_or_region:
- lesson_method(수업 방식 요지):
- activity(활동 예시):
- our_use(우리 실험/가이드 개선 아이디어):
- use_policy: evidence_of_practice / teaching_activity
- privacy_risk / minor_data_risk: (보통 high)
- requires_anonymization: true
- update_queue 등록: school_practice / teaching_activity / source_lead

## 원칙
- ★교실·아동 얼굴이 나오는 자료는 영상 직접 사용 금지 → **재구성 카드**.
- '한 학교 사례' / '한 수업 사례'로 표현. 전체 흐름처럼 과장 금지.
- 교사 개인 특정 정보 노출 금지.
