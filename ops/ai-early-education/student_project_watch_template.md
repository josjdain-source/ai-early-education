# Student Project Watch — 학생 프로젝트/대회/동아리 기록 (축E)

> 학생 사례 = **아이들이 AI로 무엇을 만드나(실제 사용).** 사실 근거 아님.
> ★★미성년자 보호 최우선. 이름/얼굴/학교 특정 금지. 재구성 카드 우선.
> 파일명 예: `student_project_watch/2026-07-05_<slug>.md`

## 사례 1
- context(대회/동아리/수업, 특정 금지 → '한 학교 대회'):
- url / date / country_or_region:
- what_students_made(무엇을 만들었나, 일반화):
- age_band(연령대만, 개인 특정 금지):
- why_interesting(부모에게 주는 의미):
- use_policy: student_activity_example
- **privacy_risk: high / minor_data_risk: high**
- **direct_quote_allowed: false / visual_use_allowed: false / requires_anonymization: true**
- rights_confirmed(작품 이미지 권리 확인): 예/아니오 → 아니오면 사용 금지
- update_queue 등록: student_activity_example / source_lead

## 원칙 (필수)
- 학생 이름·얼굴·학교가 특정되는 자료는 **영상 직접 노출 금지**.
- 공개 보도라도 미성년자 개인정보는 흐림/일반화.
- 사례는 '한 학교 사례'/'한 수업 사례'로. **단일 사례를 전체 흐름처럼 과장 금지.**
- 학생 성과 과장 금지. 작품 이미지는 **권리 확인 전 사용 금지.**
- 영상에는 직접 캡처보다 **재구성 카드**(익명·일반화)를 우선.
