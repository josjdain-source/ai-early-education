# YouTube Watch — 개별 영상 기록 (축3)

> 유튜브는 **사실 근거가 아니다.** 설명 방식·부모 질문·영상 주제·대본 구조·반응만 본다.
> 사실 주장은 반드시 축1(공식)에서 별도 확인. do_not_use_as_policy_fact=true.
> 파일명 예: `youtube_watch/2026-07-05_<slug>.md` (여러 편이면 반복 블록)

## 영상 1
- channel_name:
- video_title:
- video_url:
- publish_date:
- country_or_region:
- speaker_type: teacher / parent / researcher / company / creator / unknown
- main_claim: (핵심 주장 한 줄)
- parent_question: (이 영상이 답하는 부모 질문)
- explanation_style: (후킹·비유·화면구성 등 어떻게 쉽게 설명했나)
- useful_for: hook / structure / example / objection / topic (해당 다중 선택)
- fact_claims_need_verification: true/false (사실 주장 있으면 true → 축1 확인 대상)
- do_not_use_as_policy_fact: true (고정)
- monetization/상업성: (상품·강의 홍보 있으면 표기, 근거로 쓰지 않음)
- 우리 활용 메모: (5분 영상/쇼츠/대본에 어떻게 참고할지)
- update_queue 등록: video_topic / explanation_pattern / parent_question 중 택

## 사용 원칙 (재확인)
- 유튜브 화면/썸네일을 공식 근거처럼 쓰지 않는다.
- 영상 주장을 사이트 사실로 자동 반영하지 않는다.
- 인용 시 '유튜브 창작자 의견'임을 명시, 저작권상 짧은 참고만.
