# 1편 영상 싱크 검증 — 검수 통과 음성 고정 버전 (tail_audio_sync)

> 긴급 정정 반영: **컷별 TTS 재생성 중단.** 사람이 검수해 좋다고 한 음성을 그대로 쓰고, 화면 컷만 내레이션에 맞춤.

## 원칙 (고정)
1. 검수 통과 TTS는 재생성하지 않는다.
2. 싱크가 안 맞으면 음성을 다시 만들지 말고 화면 컷 타이밍을 음성에 맞춘다.
3. 구간은 silencedetect로 나누되 새 TTS 생성은 하지 않는다(음성 자르지 않음).
4. 컷별 TTS는 마지막 수단.
5. TTS 재생성 전 사용자 승인 필수.

## 기준
- 기준 음성: **assets/audio/door-sign-short-01.tail2s.wav** (38.29s, 2.8s tail 포함, audio_review_status=usable)
- **새 TTS 생성 여부: false** (MeloTTS 미실행, 컷별 wav 생성 없음)
- 문장 경계: ffmpeg `silencedetect=noise=-34dB:d=0.30` 으로 추출(분석만, 음성 미변형)
- 렌더: content/video/render_ep01_tail_audio_sync.py
- debug: assets/video/ep01_door_sign_tail_audio_sync_debug.mp4
- fixed(후보): assets/video/ep01_door_sign_tail_audio_sync_fixed.mp4
- 영상 길이: 40.30s = 오디오 38.29s + 2초 여유 (음성 잘림 없음, -shortest 미사용)
