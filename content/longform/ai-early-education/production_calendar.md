# AI 조기교육 데일리 관측소 — 정기 발행 캘린더

- 작성일: 2026-07-03
- 원칙: **매일은 관측(비공개), 공개 발행은 주 2~3회 + 품질 기준 통과 시.** 매일 무조건 공개하지 않는다.

## 1. 주간 리듬 (내부 작업 기준)
| 요일 | 내부 작업 | 공개 발행? |
|---|---|---|
| 월 | 이번 주 관찰 포인트 정리 (daily watch 시작) | 비공개 |
| 화 | 국가별 변화 요약 (update_queue 정리) | 비공개 |
| 수 | 부모 FAQ/오해 정리 초안 | 조건부(품질 통과 시) |
| 목 | **5분 정밀 영상 대본** 작성 | 비공개(대본) |
| 금 | **주간 브리핑** 완성 | ✅ 공개(주력) |
| 토 | 출력자료/가정 실험 콘텐츠 | 조건부 |
| 일 | update_queue 정리 + 다음 주 계획 | 비공개 |

> 매일 = 관측(daily_watch 기록). 공개 발행 목표 = **주 2~3회**(금 브리핑 고정 + 5분 영상 + 선택 1).

## 2. 콘텐츠 흐름 (파이프라인)
```
새 자료 발견
  → daily_watch 기록(비공개)
  → update_queue 등록(영향분석·출처·확인일)
  → 주간 브리핑에 반영(금)
  → 5분 정밀 영상 소재로 승격(목 대본→촬영/렌더)
  → 홈페이지 필요 시 업데이트(사람 승인 후에만)
```

## 3. 5분 정밀 영상 (주 1편 목표)
- 길이 **4분 30초 ~ 5분 30초**
- 각국 공식 자료 기반, 홈페이지 `source-library.html`와 연결
- 부모용 쉬운 말, 단정 금지(확인일·출처 화면 표기)
- 화면: 세계 지도 / 비교표 / 정책 카드 / **출처 카드**
- 쇼츠는 이 5분 영상에서 **30초 클립으로 추출**(파생물, 원본 아님)
- 대본 위치: `content/longform/ai-early-education/episode_YYYYMMDD_5min.md` (작성 시)

## 4. 발행 품질 기준 (통과해야 공개)
- [ ] 출처 URL + 확인일 명시
- [ ] 공식 발표 또는 2개 이상 교차확인(단일 보도 단독 ❌)
- [ ] 부모 행동으로 번역됨(그래서 집에서 뭘?)
- [ ] 기존 사이트 결론과 충돌 없음(있으면 update_queue로)
- [ ] 단정·공포·특정 서비스 추천·상품 없음
- [ ] 사람 승인(공개·사이트 반영 건)

## 5. 발행 채널
- 주간 브리핑: 홈페이지(향후 섹션) + 블로그/소셜 요약
- 5분 영상: 롱폼 채널 → 쇼츠 클립 파생
- 홈페이지 반영: 승인된 update_queue 항목만
