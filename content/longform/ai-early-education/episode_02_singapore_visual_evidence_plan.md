# UQ-02 싱가포르 5분 영상 — 시각 근거(Evidence) 세트 계획

> 목적: 대본(주장)에 **싱가포르 공식 자료 화면(증거)**을 박아 설득력을 높인다.
> 원칙: **관광 이미지 아님, 교육 근거 자료 중심.** 설명 70 / 증거 30. "싱가포르=정답" 아님 — 자료 위에도 "참고 사례" 톤.
> 상태: 계획서(초안). 영상 제작·캡처·업로드·사이트 수정 없음. 사람 승인 후 실제 캡처.

## 0. 캡처 원칙 (중요)
- 공식 페이지는 **URL 바가 보이게** 캡처(진짜 자료임을 증명). 컷 길이 2~4초.
- **문구 왜곡·과장 크롭 금지.** 원문 맥락 유지, 핵심 키워드만 하이라이트(밑줄/형광).
- 각 컷 하단에 **출처 카드**(기관명 + 확인일 2026-07-04 + URL). "정책은 변동 가능" 상시 톤.
- 캡처 시점에 **문구 재확인**(정책 페이지는 갱신될 수 있음). 사라진 문구는 사용 금지.
- 저작권: 공식 공개 페이지의 짧은 인용 컷 + 명시적 출처 표기(해설 목적). 로고 단독 강조/변형 금지.

## 2. 필수 증거 컷 (Essential) — 9컷 (E1·E2·E2b·E3·E4·E5·E6·E7·E8)

### E1. 싱가포르 MOE — AI in Education (도입 직후)
- 타임코드: 0:20~0:30
- 보여줄 것: `moe.gov.sg` AI in Education 페이지(URL 바 포함) + 싱가포르 국기 코너
- 강조 문구: "Artificial Intelligence in Education" (MOE 공식)
- 필요 이유: "이건 내 의견이 아니라 정부 자료"임을 첫 증거로 각인
- 출처: https://www.moe.gov.sg/education-in-sg/educational-technology-journey/edtech-masterplan/artificial-intelligence-in-education

### E2. ★학생+학부모 연령별 프레임 (핵심 증거) — 제목 문구 확인됨(2026-07-04)
- 타임코드: 0:35~0:50
- 보여줄 것: MOE 국회답변 제목 "Developing Age-progressive Framework for **Responsible Use of GenAI for Students and Parents** across Educational Levels"
- 강조 문구: **"Students and Parents"** + "Age-progressive"
- 필요 이유: 이 영상의 심장 — "부모까지 포함"을 눈으로 증명(우리 주장의 근거)
- 검증: ✅ 제목에 "Students and Parents" 실제 존재(직접 확인). 본문은 grade별 세부 문장은 없음 → 세부는 E4 참조.
- 출처: https://www.moe.gov.sg/news/parliamentary-replies/20251104-developing-age-progressive-framework-for-responsible-use-of-genai-for-students-and-parents-across-educational-levels

### E2b. ★부모용 공식 가이드 실물 (부모 포함의 결정적 증거)
- 타임코드: 0:50~1:00
- 보여줄 것: E2 답변 본문이 언급하는 **"A Parent's Guide to Generative AI tools for Learning"** (Parents Gateway 제공)
- 강조 문구: **"A Parent's Guide to Generative AI"**
- 필요 이유: "부모까지 넣었다"가 구호가 아니라 **실제 부모용 가이드 문서로 존재**함을 증명 — 가장 강력
- 검증: ✅ E2 답변 본문에서 해당 부모 가이드·Parents Gateway 언급 확인(2026-07-04). 캡처 시 Parents Gateway 원문 페이지 URL 확보할 것.
- 출처: E2 답변 본문 내 언급 / (캡처 대상) Parents Gateway 해당 가이드 페이지 — 캡처 전 실제 URL 확인 필요

### E5. AIEd 윤리 원칙 (책임 사용 구간) — ⚠️ 캡처 시 원문 확인 필수
