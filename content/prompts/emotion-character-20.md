# 사진 한 장으로 감정 캐릭터 20종 만들기 프롬프트

> 아래 프롬프트는 복사해서 사용할 수 있습니다. 사진/캐릭터 이미지 1장을 GPT에 올린 뒤 붙여넣으세요.

## 프롬프트

당신은 아이와 부모가 함께 사용할 수 있는 따뜻한 감정 캐릭터 시트를 만드는 일러스트레이터입니다.

첨부한 사진 또는 캐릭터 이미지를 참고해서, 같은 인물/캐릭터 느낌을 유지한 채 20가지 감정 표현을 가진 2D 캐릭터 시트를 만들어 주세요.

목표:
아이와 함께 만드는 문패, 스티커, 열쇠고리, 감정 카드에 사용할 수 있는 귀엽고 안전한 감정 캐릭터 이미지입니다.

스타일:
- 따뜻하고 부드러운 2D 일러스트
- 너무 사실적이지 않게
- 아이와 부모가 함께 보기에 편안한 분위기
- 둥근 선, 밝은 색감
- 과한 공포/분노 표현 금지
- 실존 캐릭터나 유명 캐릭터 스타일 모방 금지
- 특정 브랜드/작품 스타일 언급 금지

출력 형식:
- 5열 × 4행 고정 그리드
- 총 20개 감정 캐릭터
- 각 칸은 같은 크기
- 번호 1, 2, 3 같은 순번은 절대 넣지 마세요
- 각 칸 안에는 숫자 배지를 넣지 마세요
- 자르기 쉽게 캐릭터가 칸 중앙에 오게 해주세요
- 배경은 단순하고 깔끔하게
- 각 캐릭터는 나중에 개별 이미지로 잘라 쓸 수 있어야 합니다

20가지 감정:
좋아 / 신났어 / 그냥 그래 / 슬퍼 / 속상해 / 짜증나 / 화났어 / 삐졌어 / 걱정돼 / 피곤해 / 혼자 있고 싶어 / 자랑하고 싶어 / 울고 싶어 / 실망했어 / 답답해 / 말하기 싫어 / 머리가 복잡해 / 무서워 / 뿌듯해 / 웃겨

중요:
- 아이 얼굴을 과하게 사실적으로 복제하지 말고, 캐릭터화해 주세요.
- 개인정보가 보이는 요소는 넣지 마세요.
- 학교명, 이름표, 전화번호, 주소 같은 정보는 표현하지 마세요.
- 아이가 불쾌해 보이거나 무섭게 보이는 표현은 피해주세요.
- 감정은 부드럽고 이해하기 쉽게 표현해 주세요.
- 결과물은 부모와 아이가 함께 확인하는 만들기 활동용입니다.

## 안전 안내
- 아이 사진을 사용할 때는 부모와 함께 확인해 주세요.
- 이름·학교·전화번호·주소처럼 개인정보가 보이는 이미지는 사용하지 않는 것이 좋습니다.
- 생성된 이미지는 아이와 부모가 함께 확인한 뒤 출력해 주세요.
- 실존 캐릭터나 유명 캐릭터 스타일을 따라 해달라고 요청하지 마세요.
