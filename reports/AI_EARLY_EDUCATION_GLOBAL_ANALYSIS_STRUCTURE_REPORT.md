- **항목(8 컬럼)**: 시작 연령 / 필수·선택 / 수업 방식 / 생성형 AI(초등) / 교사 연수 / 윤리·안전 / 부모 가이드 / 핵심 특징.
- 표 밑에 3대 공통 규칙(①아이 혼자 안 씀 ②개념·태도 먼저 ③교사 먼저) + 한국 특이점.
- 싱가포르·중국·에스토니아·한국은 국가명이 상세 페이지로 링크, 나머지 5개국은 매트릭스에 요약.

## 4. models.html 분류 방식
6개 모델 = ‘무엇을 가장 앞에 두는가’ 기준. 각 모델에 대표 국가 + 강점 + 주의.
1. **국가 플랫폼 통제형** — 싱가포르
2. **국가 전략·의무 교육형** — 중국
3. **고학년 집중·교사 우선형** — 에스토니아, 영국
4. **역량·시민성 우선형** — 핀란드, UNESCO·유럽
5. **커리큘럼 확장·규모형** — 인도, 일본, 미국
6. **가정·민간 보완형** — 미국, 한국(제도 공백으로 가정 역할 확대; ‘국가 전략’이라기보다 현실 패턴이라 명시)
- 관통 문장: “도구를 빨리 쥐여주는 나라가 아니라, 개념·태도·어른 준비를 먼저 세운 나라가 앞선다.”

## 5. 국가별 상세 페이지 핵심 요약(각 9개 질문: 요약/시작/학교/생성형AI/보호장치/강점/위험/시사점/출처)
- **싱가포르**: 국가 플랫폼 SLS 안에서 통제. 초1~3 미사용→초4 교사 감독. 범용 챗봇 대신 ‘정답 대신 힌트’ 교육전용 AI. 정부 부모 가이드 2종.
- **중국**: 가장 공격적 의무화(베이징 연 8차시)지만 초등 개방형 생성 AI 단독 사용 금지(명문). ‘많이 가르치되 어릴 땐 통제’.
- **에스토니아**: 7세 코딩(2012~)이나 생성형 AI 지급은 고교부터. 교사 3천 명 먼저 연수. 학생 데이터 비학습 계약.
- **한국**: 세계 유일 ‘AI 교과서 전국 일괄 도입→1년 내 격하·축소’. 방향이 아니라 순서(설계<속도) 문제. 제도 공백이 가정 몫을 키움(→2편·3편 연결).
