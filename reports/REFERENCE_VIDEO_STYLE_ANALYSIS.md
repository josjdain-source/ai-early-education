## 스타일 DNA(실측)
1. **일관된 스토리북/에디토리얼 카툰 톤** — 따뜻한 뮤트 팔레트, 두꺼운 선, 반실사 인물, 디테일 배경(공장·사무실·연구실), 매 컷 **액자형 비네트 테두리**.
2. **개념을 은유 일러스트로** — 반도체 전쟁=보드게임 / 권력역전=저울 / 공급부족=경매장 / 3D적층=사람이 칩 쌓기 / 한국의 기회="한국의 자리" 의자 / 과거·미래=1999↔2026 퍼즐.
3. **타이틀 배너를 장면 안에** — 큰 한글 제목이 현수막·간판으로 그림 속에 박힘(슈퍼'을'의 탄생 / 세계 최초의 대기록 / 확실한 옆자리 차지 …).
4. **데이터·라벨 내장** — 193nm→13.5nm 그래프, "2,850억 원", "2,400억 원 투자", 브랜드 로고(ASML·TSMC·Intel·SK하이닉스), 국기(한국·대만·미국).
5. **하단 굵은 자막** — 흰 글자+검은 외곽선(내레이션).
6. **반복 캐릭터** — 백발 교수, 정장 사업가, 흰가운 연구원(대략 일관).
7. 위 정지 일러스트에 **약한 켄번즈** + 내레이션 + BGM.

## 결론: **네, 만들 수 있습니다. 그리고 이게 b-roll 문제의 해답입니다.**
- 우리가 겪던 "깨끗한 뉴스 교실 영상 구하기 어렵다" 문제 → **개념 일러스트를 직접 생성**하면 원천 해결.
- SD가 한글을 못 그리는 약점은, 참고 영상도 마찬가지라 **일러스트는 텍스트 없이 생성 → 한글 타이틀/라벨을 Pillow로 오버레이**(우리가 이미 잘함)하는 구조로 간다.

## 재현 워크플로(AI교육 콘텐츠용)
1. 개념별 **은유 설계 + 일러스트 프롬프트**(고정 스타일 토큰: warm storybook editorial cartoon, thick outline, framed).
2. **로컬 SDXL/ComfyUI**가 16:9 일러스트 생성(텍스트 없이).
3. **Pillow 오버레이**: 타이틀 배너 + 라벨/데이터 + 하단 자막 + 액자 비네트.
4. 약한 켄번즈 + 내레이션 + 조립(우리 build_draft 파이프라인 재사용).
