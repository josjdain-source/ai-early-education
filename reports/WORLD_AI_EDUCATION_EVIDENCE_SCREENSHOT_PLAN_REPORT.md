# 세계 AI교육 본편 — 증거 스샷 삽입 방식 반영 보고

- 작성일: 2026-07-04
- 성격: **제작 구조 변경(설계·계획).** 실제 스샷 다운로드/캡처/렌더/업로드/HTML 없음.

## 1. 왜 스샷 증거자료 방식을 추가했나
- 해외 사례를 **우리가 새로 그린 이미지로만** 설명하면, 부모 입장에서 "그래서 진짜 그런가?"의 근거가 약하다.
- 해외 유튜브/뉴스/공식 화면 스샷을 **짧게 인용**하면, 주장 뒤에 **영수증**이 붙어 신뢰도가 오른다.

## 2. 기존 "전부 직접 이미지" 방식의 한계
- 톤은 통일되지만 **출처가 없음** → 해외 사례라는 주장에 증거가 없음.
- 인포그래픽만으로는 "정책 설명"에 머물고, "실제로 세계가 이렇게 한다"는 증거 제시가 안 됨.

## 3. 새 3단 화면 구조
각 나라 = ①인포그래픽(우리 제작) → **②증거 스샷(해외 화면 인용 + 출처 바)** → ③부모 해석 카드(우리).
- 비율: 증거 스샷 40% · 직접 인포그래픽 40% · 부모 해석 카드 20%.
- 증거 스샷 = 배경 아님, 3~5초 짧게, 출처 바 필수, 클립/음성 미사용.

## 4. 국가별 스샷 후보 수
- 국가별 **최소 1장**(최대 2장): 중국·미국·영국·싱가포르·한국 + 공통(국제 뉴스) = **6개 슬롯**.
- 우선순위: 공식기관 > 정부·교육부 > 공영·주요 언론 > 교육 매체 > 개인 유튜버(보조).
- ⚠️ URL·제목·게시일은 **사람이 확정**(임의 URL 생성 금지). 현재 전부 pending_source.
