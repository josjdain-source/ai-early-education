어떤 상황에서 쓰나

매일 쇼츠를 올리고 싶은데 편집에 1~2시간씩 쓰고 있다면, 이 도구는 '대본 쓰는 시간'만 남기고 나머지를 자동화합니다. 이 채널의 쇼츠들이 실제로 이 프로그램으로 만들어집니다.

무엇이 쉬워지나

컷별 큰 제목·하단 자막·내레이션 싱크·컷 전환(디졸브)·느린 줌 연출이 전부 자동입니다. 완성본은 유튜브에 바로 올릴 수 있는 mp4입니다.

제한사항

그림 자동 생성은 GPU가 있어야 합니다(없으면 이미지 수동 투입 방식). 음성은 인터넷 연결이 필요합니다. 대본(내용)은 직접 쓰셔야 합니다 — 이 상품은 '조립 자동화'이지 '내용 생성기'가 아닙니다.

🔧 설치 방법 (Windows · 출시 시 가이드 동봉)

  1. Python 설치 (python.org — 'Add to PATH' 체크)
  2. 명령창에서: pip install edge-tts
  3. ffmpeg 설치 후 Path 등록 (가이드 1장에 그림 순서 수록)
  4. zip을 원하는 폴더에 풀기 — 폴더 구조 그대로 유지
  5. (선택) 그림 자동 생성용 ComfyUI 설치 — 가이드 수록

설치가 막히면 화면 캡처와 함께 이메일 주시면 같이 해결합니다.

▶ 기본 사용 방법

  1. plans/plan_template.json 을 복사해 내 대본으로 수정
  2. 명령창에서: python shorts_builder.py 대본이름
  3. 완성 mp4: output/render/ · 컷 이미지: output/sframe/ (썸네일용)
⚠ 주의 · 자동 생성 자막·음성은 업로드 전 한 번 확인하세요. 타인의 이미지·음원을 넣을 경우 저작권은 사용자 책임입니다. 모든 PC에서 동일 동작을 보장하지 않으며, 필요 환경을 먼저 확인해 주세요. 현재는 결제 준비중입니다.
