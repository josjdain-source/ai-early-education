사진 한 장으로 감정 캐릭터 20종 만들기

아이 사진이나 캐릭터 이미지 한 장을 GPT에 올리고, 아래 프롬프트를 붙여넣으면 20가지 감정 캐릭터 시트가 만들어져요.
중요한 건 프롬프트를 한 번 복사하는 게 아니라, 원하는 결과가 나올 때까지 다시 말해보는 경험입니다.

왜 이게 AI 조기교육인가요?

아이와 함께 생성형 AI를 사용하는 과정은 단순한 만들기가 아닙니다. 아이는 원하는 결과를 떠올리고, AI에게 말하고, 나온 결과를 보고, 다시 조건을 바꿔 요청합니다. 이 반복 속에서 아이는 ‘어떻게 말해야 원하는 결과가 나오는지’를 배웁니다. 이것이 유치원 시기부터 시작할 수 있는 가장 쉬운 AI 조기교육입니다.

🧩

논리력

원하는 조건을 정리해 말해요.
💬

표현력

머릿속 이미지를 문장으로 바꿔요.
🔁

문제해결력

마음에 안 들면 다시 요청해요.
💡

통찰력

말이 바뀌면 결과도 바뀐다는 걸 배워요.
이 활동의 핵심은 만들기 결과물이 아니라, 아이가 AI에게 어떻게 말해야 원하는 결과가 나오는지 배우는 과정입니다.
아이는 사진 한 장을 넣고 끝내는 것이 아니라, 결과를 보고 다시 말하고, 조건을 바꾸고, 원하는 모습에 가까워질 때까지 AI와 대화합니다. 이 과정에서 논리력, 표현력, 문제해결력, 통찰력이 자연스럽게 자랍니다.

AI 조기교육은 코딩부터가 아닙니다. AI에게 어떻게 말해야 원하는 결과가 나오는지 경험하는 것부터 시작됩니다.

AI에게 다시 말해보기 — 아이가 따라 하는 재요청 문장

프롬프트로 한 번에 끝내지 말고, 나온 결과를 아이와 함께 보며 원하는 모습이 될 때까지 다시 말해 보세요. 아래 문장을 아이가 직접 소리 내어 말해보게 하면 돼요. 이 재요청이 1편의 진짜 핵심입니다.

표정·기분 바꾸기
“웃는 표정으로 바꿔줘”“화난 표정도 보여줘”“졸린 표정도 만들어줘”
더 예쁘게·스타일
“더 귀엽게 만들어줘”“색을 더 밝게 해줘”“좀 더 동그랗게”
정리·형식
“번호는 빼줘”“5열 4행으로 정리해줘”“배경은 깔끔하게”
다시 시도
“마음에 안 들어, 다시 해줘”“방금 게 더 좋았어”

정답은 없어요. 아이가 “이건 마음에 안 들어, 이렇게 바꿔줘”라고 말할 수 있게 되는 것 자체가 목표예요.

감정 스티커 20개

사진 한 장으로 이런 20가지 감정 스티커가 만들어져요. 마음에 드는 표정을 골라 문패·키링·감정 카드에 써 보세요.

사용 방법

  1. 사진 또는 캐릭터 이미지를 준비해요.
  2. GPT(이미지 생성)에 사진을 올려요.
  3. 아래 프롬프트를 복사해서 붙여넣어요.
  4. 감정 캐릭터 20종 시트가 나오면, 원하는 표정을 골라 문패로 출력해요.

GPT 프롬프트

당신은 아이와 부모가 함께 사용할 수 있는 따뜻한 감정 캐릭터 시트를 만드는 일러스트레이터입니다. 첨부한 사진 또는 캐릭터 이미지를 참고해서, 같은 인물/캐릭터 느낌을 유지한 채 20가지 감정 표현을 가진 2D 캐릭터 시트를 만들어 주세요. 목표: 아이와 함께 만드는 문패, 스티커, 열쇠고리, 감정 카드에 사용할 수 있는 귀엽고 안전한 감정 캐릭터 이미지입니다. 스타일: - 따뜻하고 부드러운 2D 일러스트 - 너무 사실적이지 않게 - 아이와 부모가 함께 보기에 편안한 분위기 - 둥근 선, 밝은 색감 - 과한 공포/분노 표현 금지 - 실존 캐릭터나 유명 캐릭터 스타일 모방 금지 - 특정 브랜드/작품 스타일 언급 금지 출력 형식: - 5열 × 4행 고정 그리드 - 총 20개 감정 캐릭터 - 각 칸은 같은 크기 - 번호 1, 2, 3 같은 순번은 절대 넣지 마세요 - 각 칸 안에는 숫자 배지를 넣지 마세요 - 자르기 쉽게 캐릭터가 칸 중앙에 오게 해주세요 - 배경은 단순하고 깔끔하게 - 각 캐릭터는 나중에 개별 이미지로 잘라 쓸 수 있어야 합니다 20가지 감정: 좋아 / 신났어 / 그냥 그래 / 슬퍼 / 속상해 / 짜증나 / 화났어 / 삐졌어 / 걱정돼 / 피곤해 / 혼자 있고 싶어 / 자랑하고 싶어 / 울고 싶어 / 실망했어 / 답답해 / 말하기 싫어 / 머리가 복잡해 / 무서워 / 뿌듯해 / 웃겨 중요: - 아이 얼굴을 과하게 사실적으로 복제하지 말고, 캐릭터화해 주세요. - 개인정보가 보이는 요소는 넣지 마세요. - 학교명, 이름표, 전화번호, 주소 같은 정보는 표현하지 마세요. - 아이가 불쾌해 보이거나 무섭게 보이는 표현은 피해주세요. - 감정은 부드럽고 이해하기 쉽게 표현해 주세요. - 결과물은 부모와 아이가 함께 확인하는 만들기 활동용입니다.
